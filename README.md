# 01_countries 
A simple web app that fetches and displays information about countries using the RestCountries API.

## About
This project is part of my JavaScript learning journey.  
It fetches data from the [RestCountries API](https://restcountries.com/) and displays basic information such as the country's flag, name, and other details when you click on a country.

## Features
- Fetches real-time data from the RestCountries API
- Displays flag, name, and official name
- Click to toggle extra information about a country
- Responsive and simple design

## Tech Stack
- HTML  
- CSS  
- TypeScript  
- Vite  
- REST API (RestCountries)
